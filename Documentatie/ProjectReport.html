Introduction

Toyr este o aplicatie web ce reprezintă un magazin online de jucări și permite utilizatorilor să găsească și să cumpere jucării aparținând unor anumite categorii și evenimente. Utilizatorul cu status de admin poate adăuga categorii, produse si poate planifica discount-uri, promoții si evenimente cu produse.

Sistemul de management al codului-sursă

Sistemul de gestionare al codului-sursă este Github. Fiecare membru al echipei este colaborator.
S-a lucrat pe un singur branch (Master), fiecare personă dând commit pentru fiecare task sau bug rezolvat cu succes.

Tehnologii folosite

  • Procesare frontend: JavaScript + Fetch API
  • Procesare backend: PHP
  • Baza de date: MYSQL și PhpMyAdmin
  • Server Web: XAMPP Apache

Servicii web create

  • StoreManagement
  • UsersMangement

Tasks

  • Container-ul pentru un produs ce trebuie afisat intr-o lista de produse

  • Afisarea datelor unui produs cu discount-uri si promotii
